Wormbin Activity Monitor

tracks Temperature, Humidity, and Barometric Pressure over time. The goal is to provide a better worm habitat by monitoring and maintaining temperature at a consistent and optimal temperature band to increase vermicast production and population. The monitoring objectives will focus on: A.) consistent worm pile temperature and B.) consistent Wormbin humidity level to reduce wall moisture. This project will be implemented in 3 phases: 1.) Sensor package to automatically measure and send the readings to a central storage point. 2.) Software application to automatically store and display measurements based on User inquiries. 3.) Deploying that software application to a hosted software vendor such as Google Cloud Compute or Amazon AWS.

Sensor package(BME280, DS18B20) logging measurements to Influxdb via MQTT

The main sensor package is comprised of a microcontroller (ESP8266) managing sensors using the i2c protocol. The main benefit of pairing the ESP8266 with i2c sensors is low cost, excellent documentation and wide commercial availability. Microcontroller code was written in C and various sensor libraries depend heavily on open source software. All Arduino sketches are available by private inquiry.
